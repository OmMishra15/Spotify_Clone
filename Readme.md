## Spotify Clone
A Responsive, Interactive and Easy-to-use Music Platform Built With HTML,CSS and Javascript

![Project Preview]
(img/Preview.png)

## Key Features
1. **Dyanamic Music Library :** Songs and albums are dynamically added using 'songs.json'.
2. **Responsive Interface :** Made Responsive for all devices.
3. **Custom Media Controls :** Fully functional Play, Pause, Next and Previous Song buttons.
4. **Responsive Seekbar and Interative Volume Management:** Real time Progress Tracking of the Song and Beautiful management of volume Through the App only 

## How To Run Locally
1. git clone
2. Open the folder in VS Code. 
Right-click index.html and select "Open with Live Server".
